Diagram Skema Wiring Bench Power Supply

1. Jalur Power & Standby

  • PS_ON (Hijau): Ke saklar → GND.
  • PWR_OK (Abu): LED + resistor 220Ω → GND.

2. Output Tetap

  • 3.3V → Oranye → Terminal Merah #1
  • 5V → Merah → Terminal Merah #2
  • 12V → Kuning → Terminal Merah #3
  • GND → Hitam → Terminal Hitam

PENTING!

  1. Beban Dummy: Jika tegangan 12V tidak stabil, beri beban (resistor 10 Ohm 10W) pada jalur 5V.
  2. Isolasi: Gunakan Heat Shrink pada semua sambungan solder untuk menghindari korsleting dengan casing metal.
  3. Fuse: Pilih nilai fuse sedikit di bawah arus maksimal Buck Converter (misal: Buck 5A, gunakan fuse 4A).
  4. Grounding: Pastikan kabel hitam tebal yang digunakan untuk jalur arus cukup besar agar tidak panas saat beban tinggi.

Wiring Guide: DIY Bench Power Supply

Modifying a Computer ATX PSU to Adjustable Output

Bench Power Supply Wiring Diagram

1. Power & Standby (Mandatory)

  • PS_ON (Green): Connect to toggle switch → GND.
  • PWR_OK (Gray): LED + 220Ω resistor → GND.

2. Fixed Outputs

  • 3.3V → Orange → Red Binding Post #1
  • 5V → Red → Red Binding Post #2
  • 12V → Yellow → Red Binding Post #3
  • GND → Black wires → Main Black Binding Post

3. Adjustable Output

Meter OutOutput -Thick Red

Important Notes

  1. Dummy Load: If the 12V voltage is unstable, add a load (10 Ohm 10W resistor) to the 5V line.
  2. Insulation: Use Heat Shrink on all solder joints to prevent shorting with the metal casing.
  3. Fuse: Select a fuse rating slightly below the maximum current of the Buck Converter (e.g., if you want to buck 5A, use a 4A fuse).
  4. Grounding: Make sure the thick black wire used for the current path is large enough to prevent overheating under high loads.
